# Private SMP Server Info Site

A React + Vite + Tailwind website documenting the mods and datapacks for a private Minecraft SMP server.

---

## Features

- **Tabbed layout** for easy browsing
- **Card-based sections** for each mod/datapack
- **Tailwind CSS** styling with shadcn/ui components
- **Responsive design** for desktop and mobile

---

## Demo

[\[SMP-Site\]  ](https://knightvava.github.io/smp-site/)

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- shadcn/ui (Card, Tabs, Badge)
- Node.js
